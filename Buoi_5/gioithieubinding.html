Gioi thiệu property binding

Dùng để in ra giá trị biến r propery của 1 thẻ html

In ra giá trị biến ra property của thẻ input
In ra gia tri bien ra property src của thẻ img

Gioi thiệu event binding

Dùng để bắt sự kiện tương tác của người dùng (click, change)

Gioi thieu two-way data binding

Tự động đồng bộ dữ liệu 2 chiều giữa code js và html

Tức là khi html thay đổi giá trị thì biến trong code js tự động nhận giá trị mới

Ngược lại khi code js thay đổi giá trị biến, html tự động hiển thị giá trị mới
